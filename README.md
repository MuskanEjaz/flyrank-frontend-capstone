# Frontend AI Engineering Capstone

## Overview

This repository contains the initial setup for my FlyRank Frontend AI Engineering capstone project. It documents the planned development stack, project conventions, and AI-assisted workflow before feature implementation begins.

## Planned Stack

- React
- Vite
- JavaScript
- CSS
- Vitest and React Testing Library
- Git and GitHub
- GitHub Copilot or Claude for AI-assisted development

## Development Approach

The project will follow a specification-first workflow:

1. Define requirements and constraints before implementation.
2. Use AI to plan and generate focused changes.
3. Review all AI-generated code manually.
4. Test validation, accessibility, and edge cases.
5. Run tests and a production build before committing.

## Current Status

Project repository and development conventions are being established.
